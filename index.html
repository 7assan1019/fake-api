Interactive Guide

             

                

                    
1. Reading Data (GET)

                    
Fetches a list of items. By default, you get the first 10 items. Use pagination to get more.

                    

                


                

                     
2. Advanced Reading (Queries)

                     

                        

                            
Pagination: ?_page=...&_limit=...

                            
Get 5 products from the second page.

                            

                        

                        

                            
Sorting: ?_sort=...&_order=...

                            
Get books sorted by publication year (descending).

                            

                        

                        

                            
Filtering: ?field=value

                            
Get products from category 1 by 'GlobalTech'.

                            

                        

                     

                


                

                    
3. Writing Data (POST, PUT, DELETE)

                    
Simulates creating or updating an item. The server validates the request and returns a matching response, but **no changes are saved to the database**. This keeps the API clean for everyone.
